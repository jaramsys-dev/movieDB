# Spring boot JPA REST API

This application developed by using Spring boot MVC, Spring Data JPA with the ability to do CRUD operations & pagination.

## MoviesDB

REST APIs handles CRUD operations in this application. 

## Technology stack
-   Spring Boot
-   Spring MVC
-   Spring Data JPA
-   Swagger
-   lombok
-   flyway
-   H2

## Prerequisites
- JDK 8
- Maven
